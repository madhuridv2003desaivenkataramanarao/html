Odin Project Foundations

HTML and CSS Basics

In the foundation course, you will learn the basics of HTML and CSS. You'll understand how to structure a webpage using semantic HTML and style it using CSS for a visually appealing interface.

Git and GitHub

Version control with Git is introduced to manage your code effectively. You'll also learn how to use GitHub to host and collaborate on projects.

JavaScript Basics

You'll be introduced to JavaScript to add interactivity to your web pages. This section covers variables, functions, loops, and basic DOM manipulation.

Project: Building a Web Page

The culmination of the foundation course involves building a simple web page from scratch. You'll apply HTML, CSS, and JavaScript concepts learned in the course to this project.
